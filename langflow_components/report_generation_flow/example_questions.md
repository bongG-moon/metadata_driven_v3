# Report Generation E2E Example Questions

이 파일의 질문들은 `report_generation_flow`가 단독 E2E 업무 flow로 선택되어야 하는 예시입니다. 후속 분석을 뜻하는 "방금 결과로" 질문이 아니라, 사용자가 처음부터 완성 리포트를 요청하는 형태를 기준으로 합니다.

## Copy/Paste Questions

```text
오늘 DA공정 일일 운영 리포트 만들어줘

오늘 WB공정 기준으로 생산량, 재공, 목표 달성률을 포함한 요약 리포트 만들어줘

오늘 HBM 제품군의 생산 실적, 재공 현황, 목표 대비 차이를 리포트로 정리해줘

오늘 DA/WB공정의 주요 이상 징후와 우선 확인 항목을 포함해서 운영 리포트 작성해줘
```

## Expected Flow Intent

```text
사용자 질문
-> 리포트 목적/범위 판단
-> 필요한 데이터 항목 결정
-> 리포트 섹션 구성
-> 최종 리포트 메시지/API 응답 생성
```

나중에 state 기반 후속 리포트도 추가할 수 있지만, 현재 예시 기준은 신규 E2E 리포트 요청입니다.
